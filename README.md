# Movse
![logo](images/logo.png "Movse logo")

## 这啥
一个超级简单的多屏幕鼠标移动工具 for Mac。  
灵（chao）感（xi）来源于 [Hammerspoon] 和 [CatchMouse]。

## 安装
下载源码编译，或者[点此下载最新版本](https://github.com/tgmeng/Movse/releases/latest "最新版本")。  
将 Movse.app 拖到「应用程序」里即可。
